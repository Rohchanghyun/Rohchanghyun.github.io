## My code


```python
import pandas as pd

def find_customers(customers: pd.DataFrame, orders: pd.DataFrame) -> pd.DataFrame:
    non_order_df = customers[~customers['id'].isin(orders['customerId'])]
    non_order_df = non_order_df[['name']].rename(columns = {'name': 'Customers'})
    return non_order_df
```
