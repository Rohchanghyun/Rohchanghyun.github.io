## My code


```python
class Solution:
    def reverseWords(self, s: str) -> str:
        ans = ' '.join(s.split()[::-1])
        
        return ans
```

## other code


```python
class Solution:
    def reverseWords(self, s: str) -> str:
        return ' '.join([ch for ch in reversed(s.split()) if ch])
```
