## My code


```python
class Solution:
    def minEatingSpeed(self, piles: List[int], h: int) -> int:
        left = 1
        right = max(piles)

        while left < right:
            k = (left + right) // 2
            cur_h = 0
            for n in piles:
                cur_h += n // k
                if n % k != 0:
                    cur_h += 1
                    
            if cur_h <= h:
                right = k
            elif cur_h > h:
                left = k + 1
        return right
            
        
```
