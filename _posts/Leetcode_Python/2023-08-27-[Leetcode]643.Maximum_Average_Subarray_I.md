## My code


```python
class Solution:
    def findMaxAverage(self, nums: List[int], k: int) -> float:
        tmp = sum(nums[:k])
        avg_value = tmp
        for i in range(k,len(nums)):
            tmp = tmp + nums[i] - nums[i - k]
            avg_value = max(avg_value,tmp)
        
        return avg_value / k
            
```
