## My code


```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        find_zero = 0
        for i in range(len(nums)):
            if nums[i] != 0 and nums[find_zero] == 0:
                nums[i],nums[find_zero] = nums[find_zero],nums[i]

            if nums[find_zero] != 0:
                find_zero += 1
```
